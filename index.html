Get Familiar with computer languages

C is a powerful general-purpose programming language. It can be used to develop software like operating systems, databases, compilers, and so on. C is structural programming language

C++ can be used to develop OS, browsers, games, and so on. It supports different ways of programming like procedural, object-oriented, functional, and so on which makes it flexible

C# is a general-purpose, object-oriented programming language. It is used for developing desktop,web applications. C# is closer to C++ and C. Syntactically, it is similar to Java.

Java is used to develop desktop and mobile applications, big data processing, embedded systems, and so on. According to Oracle Java one of the most popular programming languages.

Python is a general-purpose interpreted, interactive, object-oriented, and high-level programming language. It was created by Guido van Rossum during 1985- 1990

The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by CSS and scripting languages such as JS
